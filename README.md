 📚 BookWorm

A full-stack web application that allows users to explore and access books using open-source APIs and publicly available digital libraries.

---

 Features

* 📱 Responsive Design (works on mobile + desktop)
* 🔍 Search and browse books
* 📖 View book details and access PDFs
* ⚡ Dynamic navigation using JavaScript
* 🌐 API-based data fetching

---

 Tech Stack

**Frontend:**

* HTML
* CSS
* JavaScript

**Backend:**

* Node.js
* Express.js

**Database:**

* MySQL

---

 Setup Instructions

 1. Clone the repo

```bash
git clone https://github.com/YOUR_USERNAME/bookworm.git
cd bookworm
```

 2. Backend setup

```bash
cd backend
npm install
npm start
```

 3. Frontend

Just open `index.html` OR use Live Server.

---

 Environment Variables

Create a `.env` file in backend:

```
DB_HOST=your_host
DB_USER=your_user
DB_PASSWORD=your_password
DB_NAME=your_database
```
